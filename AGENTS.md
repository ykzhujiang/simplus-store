# Simplus E-Commerce Store

## Overview
Simplus brand official DTC website. Next.js 15 + Tailwind CSS + TypeScript.

## Build
```bash
pnpm install
pnpm build
```

## Dev
```bash
pnpm dev
```

## Validation
```bash
pnpm lint
pnpm build  # TypeScript + build validation
```

## Source
Application code: `src/`
Specs: `specs/`
